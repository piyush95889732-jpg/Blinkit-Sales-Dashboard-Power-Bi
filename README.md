# 🛒 Blinkit Sales Dashboard (Power BI)

## 📌 Overview
This project is an interactive **Blinkit Sales Dashboard** built in **Microsoft Power BI** using an Excel dataset. The dashboard provides valuable insights into sales performance, outlet analysis, product categories, and customer ratings through interactive visualizations.

---

## 🛠️ Tools Used
- Microsoft Power BI
- Microsoft Excel (Dataset)
- Power Query
- DAX
- Data Modeling

---

## 📊 Dashboard Features
- Total Sales Analysis
- Average Sales per Item
- Number of Items Sold
- Average Customer Rating
- Outlet Establishment Trend
- Sales by Item Type
- Fat Content Analysis
- Sales by Outlet Size
- Sales by Outlet Location
- Outlet Type Performance
- Interactive Slicers for Dynamic Filtering

---

## 📈 Key Insights
- Total Sales: **$1.20M**
- Total Items: **8,523**
- Average Sales: **$141**
- Average Rating: **3.9**
- Supermarket Type 1 generated the highest sales.
- Tier 3 outlets contributed the highest sales among all outlet locations.
- Fruits, Snacks, and Household items were among the top-selling categories.
